# Trabalho N3 - Kruskal com Cytoscape.js

Aplicação web estática para editar grafos e executar o algoritmo de Kruskal passo a passo, com log detalhado e destaque visual da MST.

## Como usar

Abra o arquivo `index.html` no navegador.

## Interações disponíveis

- Clique no painel para criar nós.
- Clique em um nó e depois em outro para criar uma aresta.
- O peso inicial da aresta é calculado pela distância euclidiana entre os nós.
- Clique em uma aresta para editar seu peso manualmente.
- Clique com o botão direito em nós ou arestas para remover o elemento.
- Clique em `Executar Kruskal` para gerar o log completo da execução.
